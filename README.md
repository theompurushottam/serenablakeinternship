# Dr. Serena Blake Therapist Website

This is a responsive therapist website for **Dr. Serena Blake, PsyD** built with **Next.js 14** and **Tailwind CSS**.

## 🚀 Quick Start

1. Install dependencies:

```bash
pnpm install # or npm install / yarn
```

2. Run the development server:

```bash
pnpm dev
```

3. Open http://localhost:3000 in your browser.

## 🏗️ Build for Production

```bash
pnpm build && pnpm start
```

The site is ready for deployment on Vercel (recommended).

## 📄 Project Structure

- `src/app` – Next.js app router pages
- `src/components` – Reusable UI components
- `tailwind.config.js` – Tailwind customization

## ✨ Features

- Responsive design closely matching [reference site](https://www.drjenniferhahm.com/)
- Hero section with "Book a Free Consult" button
- About, Services, Fees, FAQ, and Contact sections
- Clean, modular, and well-commented code
